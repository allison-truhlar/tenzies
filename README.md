## About The Project

[![Product Name Screen Shot][product-screenshot]](https://chipper-rabanadas-8fbaa6.netlify.app/)

The Tenzies app allows you to play a solo dice game with the ultimate goal of getting ten dice to display the same number. Each turn, the user can choose to "hold" dice on the current number and roll the remaining unheld die. The game ends when all ten dice match. The app tracks how many rolls the user requires to reach this goal and records the top three lowest number of rolls in a leaderboard. 

The app design is from [Scrimba](https://scrimba.com). All code was developed from scratch.
